# Emoji Reaction Picker — useState Practice

Build an interactive emoji reaction bar. Click emojis, watch counts go up, crown the winner. Your first exercise with React state.

## Quick Start

```bash
git clone https://github.com/LoisBN/fpp-reaction-picker.git
cd fpp-reaction-picker
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to see the app.
